## Installation

```bash
npm install
```

## Running the app

```bash
# application : development docker
npm run start:docker

# application : development local
npm run start:dev

# application : development pord
npm run start:pord

# db : development redis 
npm run start:redis
```

## Build

```bash
npm run build
```
